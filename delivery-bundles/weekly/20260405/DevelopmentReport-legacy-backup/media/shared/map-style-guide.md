# Sydney Map Style Guide

## Purpose

Shared visual language for the 5 short videos in `media/week_1/`.

## Base Visual Direction

- Dark premium dashboard look
- Sydney-focused, clean, modern, slightly neon
- Strong contrast between low-risk and high-risk signals
- Maps should feel editorial, not government-portal bland

## Reference Source

Borrow the visual language from:

- `dashboard/latest-report.html`
- `dashboard/hero-visual-pack.html`

## Core Palette

- Background: `#0b1020`
- Panel: `#121932`
- Accent cyan: `#33d1ff`
- Accent lime: `#b7ff3c`
- Accent amber: `#ffb547`
- Accent red: `#ff6b6b`
- Accent pink: `#ff5daa`
- Muted text: `#92a0c4`
- Map background: `#0f1731`

## Visual Mapping Rules

- Positive / cleaner signal: lime
- Neutral / contextual: cyan
- Caution / medium pressure: amber
- High pressure / stacked risk: red
- Supporting highlight / editorial callout: pink

## Map Rules

- Start with a Greater Sydney frame before zooming into highlighted precincts.
- Keep labels sparse. Only show the 3-6 names relevant to the current episode.
- Use glow or halo around highlighted areas instead of cluttered boundary styling.
- Prefer animated point pulses, soft choropleth, or selective callout cards over dense full-map labeling.

## Typography

- Sans serif similar to Inter
- Large, clean, high-contrast labels
- Minimal legend text

## Motion Suggestions

- Slow push-in from metro view to target precincts
- Soft fade between environmental layers
- Count-up stats for high-level totals
- Pulse animation on highlighted suburbs

## Output Goal

Each episode should have:

- one opening Sydney map shot
- one or two focused suburb/precinct highlight shots
- one compact chart or stat graphic
- one closing summary visual

# 首批重点 Watchlist

## 目标

先把范围收窄到政策密度高、客户容易理解、也容易做出漂亮图表的区域。

## 第一批优先 LGA 组合

### 1. Parramatta / Cumberland / Ryde

理由：

- 交通和就业节点密集
- health / education / metro 叙事强
- 适合做 `station precinct` 型机会图

首批重点 precinct：

- Westmead
- Epping
- Meadowbank
- West Ryde
- Auburn

### 2. Canterbury-Bankstown / Georges River

理由：

- rail / metro 叙事明确
- 多个 centre 的政策与市场错配可能更容易讲清
- 对 buyer's agent 也更有吸引力

首批重点 precinct：

- Bankstown
- Riverwood
- Beverly Hills
- Kogarah
- Hurstville

### 3. Inner West / Burwood / Strathfield

理由：

- 中内环、交通成熟、故事密度高
- 更容易形成“规则变化与既有低密度基底之间的反差”

首批重点 precinct：

- Ashfield
- Croydon
- Homebush
- Flemington
- Petersham

### 4. Liverpool / Campbelltown

理由：

- 西南增长走廊叙事强
- 交通、就业、增长预期和供应主题更容易同框展示

首批重点 precinct：

- Liverpool
- Glenfield
- Ingleburn
- Campbelltown
- Macarthur

### 5. Willoughby / Ku-ring-gai / North Sydney

理由：

- 北岸低密度背景明显
- 政策松绑和优质区位叠加后，容易形成强对比度故事

首批重点 precinct：

- Chatswood
- St Leonards
- Gordon
- Lindfield
- Victoria Cross

## 第一批不优先

- 全州其他非 Sydney 区域
- 噪音很高但 acquisition 意义不明显的热点 suburb 话题
- 过度依赖社交媒体热度的区域

## 第一批工作原则

- 每个 LGA 组合只保留少量高辨识度 precinct
- 每周输出优先讲 precinct，不强求一开始就下沉到单地块
- 只有在政策催化和约束判断都更清楚时，才把 precinct 升级成 site-level shortlist

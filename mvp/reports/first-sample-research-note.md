# 首批样本研究笔记

## 说明

这不是完整市场结论，而是基于当前已经真实验证并落表的数据做出的第一轮研究判断。

当前样本基础：

- `10` 个 Greater Sydney housing target council snapshot
- `15` 条 Sydney-relevant planning proposal 样本
- `14` 个重点 Sydney council 的 DA activity count
- `26,015` 条已入库 application signals

因此这份笔记适合用于：

- 判断哪些图现在值得先做
- 判断哪些区域值得继续采更深数据
- 判断哪些故事线已经能成立

不适合用于：

- 做完整 ranking
- 做 site-level 交易判断
- 做 DA 活动强弱结论

## 样本层面的第一批结论

## 1. 高目标压力最突出的样本集中在西部、西南和中西部增长走廊

在当前已验证的 10 个 council 样本中，目标值最高的一组是：

- Parramatta：`19,500`
- Liverpool：`16,700`
- Canterbury-Bankstown：`14,500`
- Ryde：`11,600`
- Campbelltown：`10,500`

这说明第一批研究最值得优先盯的，不是纯粹的高端低密区，而是：

- 中西部增长节点
- 西南增长走廊
- 交通和中心节点密集区域

## 2. 北岸样本的优势不是目标最高，而是“目标压力 + 宜居基础”组合更好

当前样本里：

- Willoughby：tree canopy `33.9%`
- Ku-ring-gai：tree canopy `43.8%`
- North Sydney：tree canopy `29.3%`
- Ryde：tree canopy `27.0%`

同时这些区域的 heat vulnerability 普遍更低。

这意味着北岸和中北样本更适合讲的，不是“目标最大”，而是：

`在较好的宜居基础上承接更多政策推动型增量。`

## 3. 西南样本已经出现很清晰的“高压力 + 高热脆弱性”叙事

当前样本里：

- Liverpool：高 heat vulnerability 约 `32%`
- Canterbury-Bankstown：高 heat vulnerability 约 `21%`
- Campbelltown：高 heat vulnerability 约 `19%`

同时这些区域住房目标都不低。

这意味着现在已经能做的一个非常有力的研究图，是：

`housing target pressure vs heat vulnerability`

这个图有两个价值：

- 对外传播性强
- 对内筛选也有用，因为它提醒我们不要只看政策增量，还要看交付环境和宜居压力

## 4. Planning Proposals 样本已经覆盖了完整的正向和负向政策路径

当前样本里已经有：

- `under assessment`
- `pre-exhibition`
- `under exhibition`
- `finalisation`
- `made`
- `withdrawn`

这点很重要，因为这说明我们后续不是只能讲“推进中的机会”，还能讲：

- 哪些 proposal 在推进
- 哪些 proposal 已兑现
- 哪些 proposal 失败或终止

这会让后续信号质量判断更稳，不会把所有政策讨论都当成真实机会。

## 5. 当前样本里已经出现一批很适合做研究 watchlist 的 Sydney council

从 proposal 样本看，值得继续加深的 council / precinct 方向包括：

- Canada Bay
- Sutherland Shire
- Inner West
- Willoughby
- Hunters Hill
- Fairfield
- Penrith
- Blacktown
- Georges River
- Bayside
- The Hills Shire

这不等于它们就是最终最好机会，但说明它们已经具备：

- 公开可追踪的 proposal 样本
- 可用于 stage analysis 的结构
- 后续适合继续叠加 housing target 和 constraints

## 6. 当前 activity 数据已经能支持一个有用的 council-level ranking

在当前已验证的 recent activity 样本里，近年记录量最高的一组是：

- Canterbury-Bankstown：`3423`
- Liverpool：`3320`
- Inner West：`2593`
- Campbelltown：`2591`
- Parramatta：`2253`
- Cumberland：`2208`

这组结果至少说明两件事：

- `Application Tracker` 不再只是理论可用，而是已经可以用于 council-level activity ranking
- 西南走廊、中西部和中内环样本已经开始出现“目标压力 + proposal + activity”可以同框的基础

但要注意：

- 这仍是 council-level 计数，不是 precinct-level activity heatmap
- 还不能直接把它当成机会排序

## 7. 当前最值得先做的研究故事线

基于现在已验证的数据，我会优先写这 3 条故事线：

### Storyline A

`Which Greater Sydney councils carry the heaviest housing target pressure?`

适合用：

- housing target ranking
- target vs existing homes

### Storyline B

`Where does growth pressure clash with heat and canopy constraints?`

适合用：

- target vs heat vulnerability
- target vs tree canopy

### Storyline C

`Which Sydney councils are already active across the planning proposal pipeline?`

适合用：

- policy stage distribution
- Sydney proposal watchlist
- positive vs withdrawn signals split

### Storyline D

`Which focus councils already show meaningful recent application activity?`

适合用：

- recent activity ranking
- target pressure vs recent activity scatter
- application scope split（ordinary vs state significant）

## 现阶段还不能讲得太满的内容

### 1. DA activity 已经升温的区域

原因：

- `Application Tracker` 结果层还没打通

### 2. 单 precinct 的确定性 site opportunity

原因：

- proposal 还没和 activity、geometry、constraints 完整拼接

### 3. 从政策到完工的完整 pipeline time lag

原因：

- 还缺稳定的 application / construction / completion 链路

# Raw: Planning Proposals

存放 `Planning Proposals Online` 的原始快照。

建议文件：

- `ppr_under-assessment_YYYY-MM-DD.html`
- `ppr_pre-exhibition_YYYY-MM-DD.html`
- `ppr_on-exhibition_YYYY-MM-DD.html`
- `ppr_post-exhibition_YYYY-MM-DD.html`
- `ppr_finalisation_YYYY-MM-DD.html`
- `ppr_made_YYYY-MM-DD.html`
- `ppr_withdrawn_YYYY-MM-DD.html`
- `ppr_proposal_<slug>_YYYY-MM-DD.html`

原则：

- 每次抓取保留原始列表页
- 详情页只抓进入 watchlist 或发生变化的条目
- 不覆盖旧快照
